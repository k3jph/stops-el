This package provides a set of macros for enforcing preconditions in your Emacs
Lisp code.


* Installation
Clone this repository or download the 'stops.el' file and place it in your load
path. Then, add the following line to your Emacs configuration:

#+BEGIN_SRC emacs-lisp
(require 'stops)
#+END_SRC

* Usage

After installing the package, you can use the guard macros in your Emacs Lisp
code to ensure various conditions are met.

#+BEGIN_SRC emacs-lisp
(defun divide (x y)
  (stops-if! (zerop y))
  (/ x y))
#+END_SRC

** API Documentation

*** stops-if!

Stops if the condition given is true.

#+BEGIN_SRC emacs-lisp
(stops-if! (integerp x))
#+END_SRC

*** stops-if-not!

Stops if the condition given is not true.

#+BEGIN_SRC emacs-lisp
(stops-if-not! (natnump x)
#+END_SRC

*** Options

All macros also take an optional argument for specifying the type of signal to
raise if the condition is met.

#+BEGIN_SRC emacs-lisp
(stops-if! (equal  x y) 'my-custom-error)
#+END_SRC
